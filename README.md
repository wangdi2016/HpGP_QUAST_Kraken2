# HpGP_QUAST_Kraken2

## Overview

This repository documents the taxonomic classification of 1011 Helicobacter pylori Genome Project (HpGP) sequences using QUAST and Kraken2.

It includes:

* Data sources and download instructions
* Analysis pipeline scripts
* Final classification results


## Repository Structure

```
├── HpGP-Kraken2/           # For Kraken2
├── HpGP-QUAST/             # For QUAST
└── README.md
```
